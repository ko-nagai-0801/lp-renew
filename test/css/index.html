About usＬｉＮＥ ＰＡＲＫについて

誰もが当たり前のことを、当たり前にできる世界へ

できること、できないこと、には必ず環境要因があると我々は考えます。
自分が正しい、ではなく様々な方々が共存していることを理解する。そして、1人1人の気持ちを共有、理解しながら共に進んでいく。
我々はそのような世界が実現されることを理想とし、まずはその歯車の1つになることを目指しています。

Service事業内容

Webサイト・ホームページ・ランディングページ、コミュニティサイトなど、コーディングを含むWeb制作。 Webデザインからバナー、チラシ、企業ロゴ、名刺などのデザイン制作。 障害福祉に係るコンサルティングやディレクション、などのサービスを主軸に展開しています。

下記、企業様のお困りごと解決のお手伝い事例も御座います。
・営業先リストの代行作成。
・データ収集とリスト化。
・Webサイトシステムのバグ調査。
・オンラインによるその他作業のお悩み相談。

ホームページ制作
